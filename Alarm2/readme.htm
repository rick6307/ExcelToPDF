在開始使用 SMSCMD 之前 必須 :

  1. Unzip smscmd.zip to c:\smscmd
  2. 至 www.sms4mail.com 註冊會員試用或購買簡訊點數
  3. 修改 sms4mail.txt 檔案 將 xxx@xxxx.xxx 換成會員email帳號 xxxxxx 換成正確密碼 請勿更動 sms4mail.txt格式 (例如: email=michael@yahoo.com&pwd=VBRGUE)
  4. 直接在 Window Command Line 輸入 smscmd 即會有指令使用說明

指令說明 :

smscmd <text> <to mobile phone number> [e]

<> : 必須要有的資料  [] : optional

例子 :
smscmd "這是少於70個中英文字的測試 This is English and Chinese test" "0935123456,09681234556"

或是

smscmd "This is pure English content less than 160 character test" "0935123456,09681234556" e

注意 !! 簡訊內容請勿放 & 因為 & 以後的簡訊內容會被截掉 且簡訊內容需用 " " 夾住

Called by VB/VC/C or any other programs

目錄需固定在 c:\smscmd.  程式呼叫 c:\smscmd\smscmd.exe
　

VB Sample (copy & paste) :

　

*******sendsms function here

Function sendsms(smstext As String, phone As String) As Variant
Dim pid As Variant
Dim sms As String

smstext = Mid(smstext, 1, 69)
sms = """" & smstext & """ " & """" & phone & """"   '簡訊內容需用 " " 夾住
pid = Shell("smscmd.exe " & sms, vbNormalFocus) 'vbHide則是隱藏執行模式
sendsms = pid

End Function

'*****Call sendsms function

Dim Totext As String
Dim Tophone As String
Dim Retcode As Variant

Totext = "放簡訊內容 最多有69個中英文字"
Tophone = "0928123456" '多個號碼用逗號隔開如 0928678901,0930123456,0935123456,0937....

Retcode = sendsms(Totext, Tophone) '當Retcode =0時則執行失敗
